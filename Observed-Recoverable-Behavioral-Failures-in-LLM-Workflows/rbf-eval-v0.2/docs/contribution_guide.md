# Contribution Guide

New tasks should be neutral, non-sensitive, reproducible, and verifiable.

Each contributed task should include:

- stable ID
- mode code
- task family
- prompt
- first-pass acceptance criteria
- retry conditions
- deterministic checks where possible
- human-judged flag where needed
- metadata indicating source type, privacy status, difficulty, and version

Avoid private conversations, employer-sensitive material, personal identifiers, and copyrighted content that cannot be redistributed.
