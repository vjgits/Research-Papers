# Annotation Rubric

For each item:

1. Determine whether the first-pass output satisfies acceptance criteria.
2. If first-pass fails, determine whether the retry output succeeds.
3. Check whether the retry prompt introduced new substantive information.
4. If retry succeeds without new information, label as RBF and assign a mode code.
5. If retry succeeds only because the user supplied new information, label as ordinary interactive correction.
6. If retry fails, label non-recoverable under available context.
7. If unclear, label ambiguous rather than forcing a code.

For artifact tasks, evaluate the artifact itself, not merely the text describing it. For citation tasks, evaluate claim-source support at the claim level.
